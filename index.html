Go-Weiqi-Canvas by josher19

Play Go (Weiqi) using Canvas

Play Go (Weiqi) online. Start with HTML version, then Canvas Version, and finally 2 player using Zappa as a backend.

Install

Front end will require browser with HTML5 Canvas Tag (such as Chrome, FireFox, Safari, or IE9).
